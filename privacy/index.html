Overview

GPT-CLIK is a Chrome extension that helps users send user-selected text from webpages to a selected AI chat service (ChatGPT, Claude, or Gemini) through a right-click menu or keyboard shortcut.

Core purpose: user can send selected text from webpages to ChatGPT, Claude, or Gemini via context menu or keyboard shortcut.

What GPT-CLIK Does

GPT-CLIK provides a simple, user-initiated workflow:

  • You select text on a webpage.
  • You trigger GPT-CLIK using the context menu or shortcut.
  • GPT-CLIK opens your selected AI service and inserts the selected text into the chat composer.

GPT-CLIK has a single purpose: sending user-selected text to a chosen AI chat service.

Data We Handle

GPT-CLIK may handle the following data:

A. User-selected website text

  • Processed only when you explicitly trigger GPT-CLIK.
  • Sent to the AI service you selected (ChatGPT, Claude, or Gemini) to provide the feature.

B. Extension preferences saved in Chrome storage

  • Selected AI provider
  • Interaction mode (for example, instant send or paste/edit)
  • Theme preference
  • Saved chat mode setting
  • Optional saved chat URL

User-selected text may be transmitted to the user-chosen AI service only when user triggers the feature.

What We Do Not Collect or Sell

GPT-CLIK does not:

  • Sell personal data
  • Use user data for advertising
  • Share user data with data brokers
  • Perform background surveillance or hidden tracking
  • Collect financial, payment, health, or authentication credentials as part of its core design

No sale of data. No advertising use. No data broker sharing. No background tracking.

No collection of health, financial, authentication, location, or web history data by default.

How Data Is Used

Data is used only to provide and improve the extension's user-facing functionality:

  • Send selected text to your chosen AI service when you request it
  • Remember your extension settings and preferences

Third-Party Services

When you use GPT-CLIK, selected text may be sent to one of these services, depending on your choice:

  • ChatGPT (OpenAI)
  • Claude (Anthropic)
  • Gemini (Google)

Once data is sent to a third-party service, that service's own terms and privacy policy apply.

Third parties: OpenAI/ChatGPT, Anthropic/Claude, Google/Gemini when user chooses them.

Data Sharing

We only transfer data when necessary to provide the extension's core function (for example, sending selected text to the AI provider you selected).

We do not transfer user data for personalized ads, data resale, creditworthiness, or lending purposes.

Data Retention

  • Preference data remains in Chrome storage until you change it, clear browser extension data, or uninstall GPT-CLIK.
  • GPT-CLIK does not maintain a separate external user database for extension settings.

User Control and Choices

You control when text is sent:

  • GPT-CLIK acts on explicit user action.
  • You can change or clear saved settings at any time.
  • You can uninstall the extension at any time.

User can edit/clear saved preferences. User can uninstall extension to remove extension data from Chrome storage.

Security

GPT-CLIK is designed to use secure HTTPS connections to supported AI destinations.

No method of transmission or storage is guaranteed 100% secure, but we aim to follow reasonable security practices.

Data transmission uses HTTPS endpoints.

Children's Privacy

GPT-CLIK is not directed to children under 13, and we do not knowingly design it for child-directed data collection.
